# DeathrollingAPP
Features
  MainMenu
    .Play button
    .Exit button
  Deathrolling Game
    .The ability to enter player names or it defaults to Player 1 and 2
    .When the submit button is pressed it will set the names and the button will Change to make the players roll numbers and names will be displayed at the top
    .another box will appear which will display the latest roll
    .the rolls will be viewed in the roll window
    .the player that reaches 0 first loses and the other player wins
    .A reset button appears which resets the game
    .A Main menu button
 
 Issues
  .The reset button does not disappear till the players start rolling again.
